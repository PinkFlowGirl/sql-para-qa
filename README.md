# SQL para QA

Projeto prático de aprendizado em SQL com foco em Quality Assurance.

## Conteúdos praticados

- Modelagem relacional
- Criação de tabelas
- Chaves primárias e estrangeiras
- Integridade referencial
- Inserts e testes
- Consultas com JOIN

## Objetivo

Praticar SQL na prática simulando um sistema de matrículasoga de Escola de Y.
